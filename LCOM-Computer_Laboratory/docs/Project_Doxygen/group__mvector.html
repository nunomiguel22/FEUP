Rotates vector using the rotation matrix.

Vector Rotation Matrix

{ cos(angle) , -sin(angle) }

{ sin(angle) , cos(angle) }

rotated v1.x = v1.x * cos(angle) - v1.y * sin(angle)

rotated v1.y = v1.x * sin(angle) + v1.y * cos(angle)
